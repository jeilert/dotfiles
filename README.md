# dotfiles for Fedora
***
### Contents
Installs the following apps:
* Google Chrome
* VS Code
* Typora
* Claude Code

##### Creates a custom app to create web apps in Chrome
Installs these custom web apps:
* Outlook
* Slack
* Claude AI
***
### Fresh Install
> Make sure you are connected to the internet before running.

1. `git clone https://github.com/jeilert/dotfiles.git ~/.dotfiles`
2. `cd ~/.dotfiles`
3. `bash install.sh`

> After the install is complete, log out and back in for all apps to appear in the app grid.
***
### Adding new web apps after install
Open the **Create Web App** app from the app grid and follow the prompts.
***

* Fedora 43 with GNOME
